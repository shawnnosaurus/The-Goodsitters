# The Goodsitters - Website Mockup

A first-pass website mockup for The Goodsitters, a Cape Town-based pet sitting and dog walking business.

## Contents

- `index.html` - Single-page website with all recommended sections
- `styles.css` - Brand styling (green #2D5A3D, cream #F5F2EB, amber #E8A838)
- `content/` - Brand brief and placeholder copy for customisation
- `docs/SEO-AND-GOOGLE-BUSINESS-GUIDE.md` - SEO and Google Business Profile checklist

## Viewing the Mockup

Open `index.html` in a browser. No build step required.

For local development with live reload:

```bash
npx serve .
```

## Customisation Checklist

1. Replace placeholder phone/WhatsApp/email in `index.html` and contact links
2. Add real testimonials from Facebook/Google reviews
3. Add founder name and story in the About section
4. Add pricing from shared Google Drive when available
5. Add logo (replace text logo in nav)
6. Connect contact form to Formspree or Netlify Forms (replace form action in index.html)

## Hosting

Deploy to any static host: Netlify, Vercel, GitHub Pages, or traditional web hosting. Upload `index.html`, `styles.css`, and any assets.

## Tech Stack

Plain HTML and CSS. No framework. Easy to edit and host.
